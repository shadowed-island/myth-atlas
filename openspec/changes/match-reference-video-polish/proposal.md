## Why

The MVP proves the world story globe interaction, but the reference video succeeds because it feels like a dense mythological star map with motion, glow, short-form storytelling, and audio feedback. This change narrows the remaining gap by adding visual density, cinematic globe layers, a tour mode, and Web Audio feedback.

## What Changes

- Increase globe visual density with decorative story clusters, emoji markers, and region-like heat fields.
- Add cinematic scan bands, arcs, pulses, and depth layers that evoke the reference video's dark luminous map.
- Add a compact right-side thumbnail wall mode while preserving accessible story selection.
- Add an autoplay tour that flies between representative stories and opens detail cards.
- Add opt-in Web Audio ambience, click, hover, and camera-flight effects.
- Extend e2e coverage for audio toggle, tour flow, and visual-density health checks.

## Capabilities

### New Capabilities

- None.

### Modified Capabilities

- `story-globe`: Enhances the story globe with reference-video-inspired visual density, motion, audio feedback, and guided tour behavior.

## Impact

- Updates the existing Three.js scene with additional procedural geometry and animation.
- Adds browser-only Web Audio utilities that remain silent until user interaction.
- Updates UI controls and responsive layout to support sound and tour controls.
- Keeps the data source static and avoids backend/CMS scope.
